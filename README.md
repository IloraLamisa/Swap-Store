# 🛍️ Swap Store

A modern e‑commerce mobile app built with **React Native** and **Expo**.  
Features onboarding, authentication, dynamic category browsing, product listings, cart, and checkout.

---

## ✨ Features
- **Auth Loading** – Skip onboarding/login for logged‑in users.
- **Login / Sign Up** – PIN‑based authentication.
- **Persistent Login** – Stored in AsyncStorage until logout.
- **Dynamic Categories** – One screen for all categories.
- **Product Listing** – Popular & On‑Sale sections.
- **Cart & Checkout** – Managed with Context API.

---

## 🛠 Tech Stack
- React Native (Expo)
- React Navigation (Native Stack)
- AsyncStorage
- Expo Font
- Ionicons
- Custom Theme (`theme.js`)
- Centralized API (`api.js`)
